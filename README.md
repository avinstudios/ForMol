# ForMol Web Browser

A modern browser-style web interface with tabs, bookmarks, and PWA support.

## Features
- Tab system
- Navigation bar
- Bookmark-ready structure
- Offline support (PWA)
- Local storage session restore
- Mobile responsive UI

## Run locally
Just open index.html

## Deploy to GitHub Pages
1. Upload all files to a GitHub repo
2. Go to Settings → Pages
3. Select branch: main
4. Done 🎉
